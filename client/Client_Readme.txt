Readme for Client Source Code

The client code for the Fruit Ninja Android app. The android app needs to connect to a computer running Fruit Ninja, and this piece of code is what the client computer will run to create a server.

	- MouseCursor - a standalone java program that opens the communication with an android phone. 
	
	- UDPTestSend - a tester java program that you can run to see if the wifi network you are using allows transmission of UDP packets.
